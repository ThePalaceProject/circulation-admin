# Jest Unit Tests

This directory contains unit tests that run using jest. The directory structure should mirror that of the `src` tree.

These are the project's tests — all of them, since the mocha/enzyme suites were rewritten here. Run them with `npm test` (or `npm run test-jest`).
To run a single test file, use `npm run test-jest-file [filename]` (or `npx jest --testPathPattern='<name>\.test'`).
